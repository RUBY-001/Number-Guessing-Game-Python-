# User Number Guessing Game (Python)
A simple Python number guessing game where the player tries to guess a randomly generated number by the computer. The program gives hints whether the guessed number is higher or lower than the target number.
This project contains multiple versions of the same game, showing different approaches to solving the same problem. Each example demonstrates different Python concepts and methods for building the game.
